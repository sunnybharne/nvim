# Neovim Snippets

A curated collection of professional code snippets for Neovim using LuaSnip.

## Prerequisites
- Neovim 0.8+
- LuaSnip plugin

## Installation
```bash
git clone <your-repo-url> ~/.config/nvim
cd ~/.config/nvim
./install.sh
nvim
```

## Usage
- Expand snippets with `<Tab>` in supported file types
- Customize snippets in `lua/snippets/`

## Available Modules
- Bicep
- Terraform
- TypeScript CDKTF
