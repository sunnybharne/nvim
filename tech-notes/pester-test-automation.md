# Pester 
Pester a testing and mocking framework for Powershell.

## Basics
- Used for
```markdown
- Unit testing
- Integration testing
- Mocking
- Whole environment testing
- Computer deployment testing
- Database configuration testing
```

- File format
```markdown
- *.Test.ps1
```

- Available functions
```markdown
- Describe
- Context
- It
- Should
- Mock
```

- Can test
```markdown
- scripts
- Functions
- Cmdlets
- Modules
```

Continue from [here](https://pester.dev/docs/quick-start#creating-a-pester-test)
